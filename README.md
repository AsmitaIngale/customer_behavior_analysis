# customer_behavior_analysis
End-to-end Customer Behavior Analysis project using Python, SQL, and Power BI with data cleaning, business analysis, customer segmentation, and interactive dashboard visualization.
# 📊 Customer Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases. The goal is to identify customer spending patterns, product preferences, subscription trends, and revenue-driving segments using Python, SQL, and Power BI.

## Technologies Used

- Python
- Pandas
- PostgreSQL / SQL
- Power BI
- Excel
- GitHub

## Dataset Information

- Total Records: 3,900
- Total Columns: 18
- Customer Demographics
- Purchase Details
- Shopping Behavior
- Subscription Information

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. SQL Business Analysis
5. Dashboard Development in Power BI
6. Business Recommendations

## Key Analysis

- Revenue by Gender
- High Spending Discount Users
- Top Rated Products
- Shipping Type Analysis
- Subscriber vs Non-Subscriber Analysis
- Customer Segmentation
- Revenue by Age Group
- Top Products by Category

## Dashboard Features

- Number of Customers
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Interactive Filters

## Key Insights

- Young Adult customers generated the highest revenue.
- Clothing category contributed the highest sales.
- Loyal customers formed the largest customer segment.
- Subscription behavior influenced purchasing patterns.
- Express shipping customers showed higher spending.

## Business Recommendations

- Improve customer loyalty programs.
- Increase subscription benefits.
- Optimize discount strategies.
- Promote top-performing products.
- Focus marketing on high-value customer segments.
